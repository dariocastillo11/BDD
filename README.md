# 📘 Quiz de Bases de Datos

Un quiz interactivo para estudiantes de Bases de Datos con 200+ preguntas.

## Características

✨ **Estilo oscuro profesional**
- Interfaz moderna y fácil de usar
- Formato celular (móvil)
- Botones para navegar y controlar

📚 **Contenidos**
- Parcial 1: Conceptos fundamentales
- Parcial 2: Transacciones, índices y optimización
- Teoría: Material de referencia completo

🎯 **Funciones**
- Progreso en tiempo real
- Feedback inmediato
- Resultado final con porcentaje
- Volver al menú en cualquier momento

## Instalación Local

```bash
pip install -r requirements.txt
python baseadatosPreguntero.py
```

## Cómo usar en Replit

1. Ve a https://replit.com
2. Crea una nueva cuenta (o inicia sesión)
3. Click en "+ New Replit"
4. Selecciona "Import from GitHub"
5. Pega el link de tu repositorio
6. Click en "Import"
7. Abre el link público para compartirlo

## Requisitos

- Python 3.7+
- Pygame 2.5.2
